# doctr-net
Set of diagnostic tools for .NET core. Helpful for old-fashioned `Console.Writeline`.

# Features

* Trace/Debug logs to ILogger
* Object Dump
* Object Internal
* Current Line Info
    * Threads
    * StackTrace
    * Modules
* Metrics
* AspNet Core Middleware
    * Endpoint Tracking
    * Metrics
* AspNet Endpoint
    * Globals Inspection
    * Global Store
